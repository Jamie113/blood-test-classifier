Per-marker Gaussian mixture model

The point. Does this marker split into natural sub-populations, or is it one tight band? Fasting glucose really does form two humps — a healthy band and a diabetic band. Sodium is one band for almost everyone. This chart shows whichever the data actually contains, with no reference range fed into the model.

What you're looking at. Each coloured curve is one sub-group the model detected. The bars are the raw count of tests at each value. Vertical lines mark where one group hands over to the next. A single curve means no detectable sub-structure for this cohort.

When it's interesting. Two well-separated curves = a genuinely bimodal marker, worth investigating clinically. Overlapping curves with a faint dip = soft sub-populations. One curve = a uniform marker for this cohort, no story to tell.

How it works — technical detail

What we fit. A Gaussian Mixture Model (GMM) is fit to each marker independently. The model assumes the observed values come from a mixture of K bell-shaped (Gaussian) components, each with its own mean µ, standard deviation σ, and mixing weight w (the weights sum to 1).

How we choose K. For each marker we fit K = 1, 2, … up to a cap of min(4, max(2, n // 5)) where n is the number of tests for that marker. We pick the K with the lowest Bayesian Information Criterion (BIC): BIC = −2·log L + k·log n, where L is the maximum likelihood and k the number of free parameters. Lower BIC = better fit, with a penalty for complexity. We only adopt a K > 1 solution when its BIC beats K=1 by at least 6 BIC units — the conventional "strong evidence" threshold — otherwise we report a single Gaussian. This stops the view from inventing groups in genuinely uniform markers.

Library and parameters. sklearn.mixture.GaussianMixture(n_components=K, n_init=5, random_state=42). Five random restarts mitigate local optima; the random seed makes results reproducible. The marker-by-marker BIC scores for the currently shown marker are listed in the per-marker Technical details below the chart.

Group boundaries and labels. Hard labels for each test come from the GMM's posterior assignments (gmm.predict), then remapped into mean-sorted order so "Group 1" is always the lowest-mean cluster. The boundary lines drawn on the chart are the x-values where adjacent components' weighted PDFs cross, found with scipy.optimize.brentq (midpoint fallback if no crossing exists). They approximate the posterior decision rule visually but do not drive the labels themselves.

Worked example

The data. Eight fictional fasting-glucose readings (mmol/L): 4.8, 5.1, 5.2, 5.4, 7.9, 8.1, 8.3, 8.6. n = 8.

1. Fit K = 1. One Gaussian over all eight points: µ = 6.675, σ = 1.568. Free parameters k = 2 (µ, σ²). log L = −14.95. BIC1 = −2·(−14.95) + 2·log(8) = 29.90 + 4.16 = 34.06.

2. Fit K = 2. With well-separated points, EM converges to:

  • Group 1: µ1 = 5.125, σ1 = 0.217, w1 = 0.500
  • Group 2: µ2 = 8.225, σ2 = 0.259, w2 = 0.500

Free parameters k = 5 (two means, two variances, one mixing weight; the second weight is fixed by 1 − w1). log L = −5.37. BIC2 = −2·(−5.37) + 5·log(8) = 10.73 + 10.40 = 21.13.

3. Apply the rule. ΔBIC = 34.06 − 21.13 = 12.93. 12.93 ≥ 6, so we adopt K = 2.

4. Label a new test. For a hypothetical reading at x = 6.5 mmol/L the weighted component densities are:

  • w1·N(6.5 | 5.125, 0.217) ≈ 1.6 × 10−9
  • w2·N(6.5 | 8.225, 0.259) ≈ 1.7 × 10−10

The posterior is the ratio: P(Group 1 | x = 6.5) ≈ 0.91, P(Group 2 | x = 6.5) ≈ 0.09. The new test is labelled Group 1.

For a dataset point well inside its cluster (e.g. x = 5.4), the posterior is decisive: P(Group 1 | x = 5.4) ≈ 1.0 to six decimal places. Tight, separated clusters give confident assignments; values near the cross-over between components produce intermediate posteriors and flag boundary cases on the Outliers tab.

Numbers above match sklearn.mixture.GaussianMixture(n_components=K, n_init=5, random_state=42) fit to the eight values.

Assumptions and limitations.

  • Each marker is treated as 1-dimensional and independent of the others (the Clusters tab handles the multivariate case).
  • A mixture of Gaussians is a good fit for many clinical markers, but heavily skewed or bounded distributions can be over-fit to multiple components.
  • With small samples (< 30 tests) the choice between 2 and 3 components can be unstable. Small-sample warnings are surfaced explicitly in the headline.
  • The model uses no clinical reference range as input. Reference lines on the chart are added afterwards purely as context.
