{% if active == "explorer" %} For each blood marker, this view shows whether your test results form one consistent range or split into several distinct groups. We fit overlapping bell curves to the data and let the shape of the results decide how many natural groups exist — a technique called Gaussian mixture modelling. Nothing is hand-labelled; the data decides. {% elif active == "population" %} This view treats each blood test as a whole — every marker at once — and groups tests with similar overall profiles. We first compress every test's full marker panel into two summary dimensions (Principal Component Analysis), then find natural clusters in that compressed map (Gaussian mixture modelling again). Tests close together on the chart have similar overall biology. {% elif active == "investigate" %} Some blood tests don't fit cleanly into any cluster. This view flags two kinds: tests on the boundary between two clusters (the model isn't sure which they belong to), and tests with an unusual overall profile that doesn't match any typical group. Both come from the cluster model on the previous tab — we just look at how confidently each test was placed and how well it fitted the typical patterns. {% elif active == "pairs" %} This view compares any two markers to see whether they move together — when one rises, does the other rise too? We summarise each pair with the Pearson correlation coefficient, a single number from −1 (move in opposite directions) through 0 (no relationship) to +1 (move together). We start you on the strongest pairing in your data. {% endif %}
